# jcal-sync-6132

A modern, interactive Hebrew/Jewish calendar web app that displays current and upcoming year's holidays, fasts, and special dates with real-time updates, user customization, and community event sharing. It integrates traditional Hebrew dates with Gregorian conversions, providing notifications, detailed explanations of each occasion, and the ability to add personal events or community-submitted observances.

## Tech Stack
Vanilla HTML/CSS/JS + Supabase

## Setup
Run `docker-compose up` to start the infrastructure.
